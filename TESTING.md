# 测试指南 — Gemini Message Navigator v3

## 安装

1. Firefox 地址栏输入 `about:debugging#/runtime/this-firefox`
2. "临时载入附加组件…" → 选择 `manifest.json`
3. 访问 https://gemini.google.com 并开始对话

## 功能测试

### 1. 导航栏出现
- 在 Gemini 中发送一条消息
- **预期**：页面右侧出现一条 **6px 宽竖线**，竖线上有一个小圆点

### 2. 悬浮展开
- 鼠标移动到右侧竖线上
- **预期**：竖线展开为 220px 宽面板，显示消息序号和预览文字

### 3. 点击跳转
- 点击任意消息标记（展开时点击整行，收起时点击圆点）
- **预期**：页面平滑滚动到该消息位置，消息高亮闪烁后消失

### 4. 多消息
- 发送多条消息
- **预期**：导航栏实时更新，圆点数量与消息数一致

### 5. 拖拽
- 鼠标按住导航栏空白区域拖动
- **预期**：导航栏跟随鼠标移动

### 6. 深色模式
- 系统切换到深色模式
- **预期**：导航栏背景和文字颜色自动适配
